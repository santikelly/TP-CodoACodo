Sitio web de la Veterinaria Patitas Sanas.
Incluye tienda en línea de artículos para mascotas.



En la página de Inicio se muestra una sección con novedades y otra una api propia sobre informacion de razas de perros (se podría agregar una bienvenida al sitio).
En la página Nosotros en una sección se describe a la veterinaría y en otra se muestra a su personal.
En la página Servicios se describe los servicios que presta la veterinaria. ( se encuentra una animacion en el titulo de servicio)
En la página Productos se muestra los productos disponibles para la venta.(se maqueto con flexbox y los productos son traidos con una api propia desde VUE)
En la página Contacto en una sección se encuentra un formulario de contacto y en otra sección los datos de contacto de las distintas sucursales de la veterinaria.


Los puntos :
El sitio contiene 5 páginas:
- Inicio
- Nosotros
- Servicios
- Productos
- Contacto 

Animaciónes: Nosotros con bootstrap, Servicios y productos (titulos) con animate.css

Páginas responsivas: 
-Inicio, Productos(utilizando medidas en porcentaje).
-Servicios, nosotros (utilizando media query)

iFrame: Se agregaron iFrame de localizacion de google para la pagina de registro

Páginas con Flexbox: se maqueto con la pagina de productos con flexbox 

Página donde se consume un json a través de una API: Inicio, Productos (.json propios Consumidos con VUE)

Formulario: 
-se integro un formulario con checkbox
-campos que no deben estar en blanco (correo electronico,edad, correo electronico y sucursal)
